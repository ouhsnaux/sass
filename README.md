# Sass

随着 `css` 文件变大，规则变复杂，越来越难以维护。
预编译器就是用来处理这个问题的。

`Sass` 提供了诸如嵌套、混入和继承等实用工具，使用 `Sass` 语法写出的代码健壮，容易维护。
使用 `Sass` 编译器将其编译为所需的 `css` 代码。
